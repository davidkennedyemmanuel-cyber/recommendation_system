# Recommendation System Project

[![Streamlit App](https://img.shields.io/badge/Streamlit-Demo-blue)](https://share.streamlit.io/davidkennedyemmanuel-cyber/recommendation_system/main/app/app.py)

## Overview
This is a **Recommendation System** built using Python, Pandas, NumPy, and Scikit-learn.  
It demonstrates a full pipeline from **Exploratory Data Analysis**, **Data Preprocessing**, **Modeling** (using TruncatedSVD), to a **live interactive Streamlit app**.

---

## Features
- Exploratory Data Analysis (EDA) on ratings dataset
- Data preprocessing and train/test split
- Matrix Factorization using TruncatedSVD
- RMSE evaluation
- Interactive recommendations via Streamlit
- Fully compatible with Windows, macOS, and Linux
